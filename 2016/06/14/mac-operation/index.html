mac中的一些常用操作

2016-06-14
  • 常用软件

编辑器方向键

文本最开始:fn + ←
文本最结尾:fn + →
行首:command + ←
行尾：command + →

PageUp:fn + ↑️️
PageDown:fn + ↓️

向前删除(backspace)：delete
向后删除(delete)：fn+delete

输入表情符号

control + command + space

触摸板操作

右键菜单

两个手指单击触摸板

文件操作

选择不连续的多个文件

command + 单击

移动文件

command + c(复制)
command + option + v(粘贴)

显示隐藏文件

$ defaults write com.apple.finder AppleShowAllFiles -bool true

接下来重启Finder。具体方法是单击屏幕左上角苹果图标，强制退出，在下拉列表中选中Finder。

窗口操作

command + control + F 最大化/还原窗口

QQ截图

control + command + A

Chrome开发者工具

command + option + I

开发工具相关

homebrew的GUI管理工具

推荐安装cakebrew

mysql

  • homebrew安装mysql无法启动的问题：参见sf.gg
  • access denied for user root@localhost using password no或者yes的问题

iterm2终端

command + T：新建选项卡
command + ←/→：切换选项卡

webstorm

整理代码：command + option + L
删除一行：command + delete
跳转行：command + L
快速修正错误：option + Enter

sublime

复制行：command + shift + D
选择所有指定单词：command + control + G

vim

  • 跳转到指定行号，在命令模式下输入行号即可，例如：:25表示跳转到25行。
  • 行首^，行尾$

搭建终极bash：zsh,iterm2,tmux

参见推酷

命令行打印的东西太多闪屏怎么办？

用管道符啊。例如：cat bidfile.txt | less

关于外接显示器分屏

command + F1可以切换是否分屏。

百度云资源下载

mac下的百度云同步盘完全是鸡肋，推荐aria2进行资源下载，注意：BaiduExporter拓展在chrome应用商店中被删除，需要开启开发者模式载入，详见：360开发文档

远程服务器文件上传和下载

推荐使用lrzsz,注意：需要在服务器上也安装这个程序。

视频播放器

推荐mpv

如何下载Google play上的apk文件

推荐使用apk-dl

一般的网线有8根，但是实际上4根线就够用了。
